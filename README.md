Sobre os Exercícios

Cada exercício foi desenvolvido com base em desafios propostos pelo curso, simulando situações reais de uso da linguagem. 
Os temas abordam desde fundamentos básicos até manipulações mais específicas, como cálculos financeiros e relatórios formatados.

Exemplo de exercício:

# Relatório de Margem de Lucro
faturamento = 45000
custo = 25000
lucro = faturamento - custo
margem_lucro = lucro / faturamento
print(f'Seu lucro foi de R$ {lucro:,.2f} e sua margem foi de {margem_lucro:.0%}')
